# Rust WebAssembly Hello World

A minimal hello-world example using Rust compiled to WebAssembly, loaded in a browser.

## Project Structure

- `src/lib.rs`: Rust WebAssembly entrypoint.
- `Cargo.toml`: Rust package configuration.
- `index.html`: Web page that loads the generated WebAssembly package.
- `pkg/`: Generated by `wasm-pack build`.

## Prerequisites

Install Rust and tools:

```bash
rustup target add wasm32-unknown-unknown
cargo install wasm-pack
npm install -g http-server
```

## Build

From this directory, run:

```bash
wasm-pack build --target web
```

This generates `pkg/` with JS glue code and the `.wasm` binary.

## Run (Simple Web Server with npm http-server)

Start server in this directory:

```bash
http-server -c-1 -p 8080 .
```

Then open:

- http://127.0.0.1:8080

You should see: `Hello, World from Rust + WebAssembly!`

## Instruction Summary

1. Install prerequisites (`rustup` target, `wasm-pack`, `http-server`).
2. Build WebAssembly package: `wasm-pack build --target web`.
3. Run local server: `http-server -c-1 -p 8080 .`.
4. Open `http://127.0.0.1:8080` in your browser.
