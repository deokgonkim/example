# Google Sheets Javascript API access example

## Install

```bash
npm ci
```

## Examples

- quickstart.js : Read `Sheet1` from `GOOGLE_SHEET_KEY` (.env)
- copysheet.js : Read `Sheet1` from `GOOGLE_SHEET_KEY` and copy it to `Sheet3`
